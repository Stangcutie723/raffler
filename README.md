The Raffler
=======
This is a system that allows you to add a list of names or whatever you'd like and choose a random winner.
<br>

Platform: Ruby with Rails 3.2.12<br>

Dependencies:<br>

  jquery-rails<br>
  rails (= 3.2.12)<br>
  sass-rails (~> 3.2.3)<br>
  sqlite3<br>
  uglifier (>= 1.0.3)<br>
